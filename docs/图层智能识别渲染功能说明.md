# 图层智能识别渲染功能说明

## 功能概述

本系统实现了图层文件的智能识别和自动渲染功能，通过文件名和关联文件自动识别图层类型，并应用相应的渲染样式。

## 实现方式

### 1. 智能识别

系统通过以下方式识别图层类型：

#### 文件名识别
- **关键词匹配**：文件名中包含特定关键词（如"高速"、"国道"、"省道"等）
- **道路编号模式**：支持 G/S/X/Y/C 开头加数字的模式
  - `G1` → 高速公路
  - `S256` → 省道
  - `X012` → 县道
  - `Y005` → 乡道

#### 关联文件识别
- 自动读取 `.id` 和 `.dat` 文件内容
- 通过文件内容关键词进一步确认图层类型

### 2. 渲染样式配置

#### 线状图层 - 道路
| 图层类型 | 颜色 | 宽度 | 样式 |
|---------|------|------|------|
| 高速公路 | 绿色 `#22c55e` | 4px | 实线 |
| 国道 | 红色 `#dc2626` | 3.5px | 实线 |
| 省道 | 紫色 `#a855f7` | 3px | 实线 |
| 县道 | 橙色 `#f97316` | 2.5px | 实线 |
| 乡道 | 灰色 `#94a3b8` | 2px | 实线 |
| 一般市政道路 | 深灰色 `#64748b` | 1.5px | 实线 |
| 铁路 | 深灰色 `#1f2937` | 2.5px | 虚线 `8,4` |
| 地铁 | 紫色 `#7c3aed` | 2.5px | 虚线 `5,5` |

#### 面状图层 - 水域
| 图层类型 | 填充颜色 | 边框 | 透明度 |
|---------|---------|------|--------|
| 湖泊 | 海蓝色 `#3b82f6` | 深蓝色 | 0.5 |
| 大海 | 深蓝色 `#2563eb` | 更深蓝色 | 0.6 |
| 水库 | 海蓝色 `#3b82f6` | 深蓝色 | 0.5 |

#### 面状图层 - 地形
| 图层类型 | 填充颜色 | 边框 | 透明度 |
|---------|---------|------|--------|
| 草地 | 绿色 `#22c55e` | 深绿色 | 0.4 |
| 山体 | 深绿色 `#166534` | 深绿色 | 0.5 |

### 3. 代码实现

#### 后端新增文件
- `backend/app/utils/layer_type_config.py` - 图层类型智能识别配置

#### 后端修改文件
- `backend/app/services/mapinfo_service.py` - 集成智能识别功能

#### 前端修改文件
- `frontend/src/renderer/components/Map/MapInfoLayer.tsx` - 优化样式应用逻辑

### 4. 道路连续绘制

所有道路均使用实线（`dashArray = ""`）进行连续绘制，确保道路的连续性和平滑性：
```typescript
{
  lineCap: 'round',   // 圆角线帽
  lineJoin: 'round'   // 圆角连接
}
```

### 5. 铁路样式

铁路使用虚线样式模拟枕木效果：
```typescript
{
  strokeColor: '#1f2937',
  strokeWidth: 2.5,
  dashArray: '8,4'  // 虚线样式
}
```

## 使用示例

### 文件命名建议

```
G1_北京上海高速.tab     # 识别为高速（绿色）
G105国道.tab           # 识别为国道（红色）
S256_省道.tab          # 识别为省道（紫色）
X012_县道.tab          # 识别为县道（橙色）
Y005_乡道.tab          # 识别为乡道（灰色）
市政道路.tab           # 识别为一般道路（深灰色）
京沪铁路.tab           # 识别为铁路（深灰色+虚线）
太湖_湖泊.tab          # 识别为湖泊（海蓝色）
```

### API 调用

```python
from app.utils.layer_type_config import get_layer_style_by_filename

# 获取样式
style = get_layer_style_by_filename('G1_高速道路', geom_type='line')
# 返回: {'stroke': True, 'strokeColor': '#22c55e', 'strokeWidth': 4.0, ...}
```

## 配置文件位置

- **图层样式配置**: `backend/app/utils/layer_type_config.py`
- **MapInfo解析服务**: `backend/app/services/mapinfo_service.py`
- **前端渲染组件**: `frontend/src/renderer/components/Map/MapInfoLayer.tsx`

## 扩展说明

如需添加新的图层类型或修改样式，请编辑 `layer_type_config.py` 中的以下内容：
1. `LayerCategory` 枚举 - 添加新类型
2. `FILENAME_KEYWORDS` - 添加识别关键词
3. `LAYER_STYLES` - 定义样式配置

## 注意事项

1. 文件名识别优先级高于关联文件识别
2. 置信度评估：完全匹配 > 前缀匹配 > 包含关键词
3. 道路编号模式（G/S/X/Y/C）具有最高置信度
4. 铁路虚线样式仅适用于线状图层
