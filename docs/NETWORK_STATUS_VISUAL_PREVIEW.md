# 网络状态指示器 - 视觉预览

## 设计理念：航图雷达美学

网络规划工具处理"连接"和"信号"，使用雷达风格视觉元素最合适不过。

## 组件预览

```

│                                         │
│  ● 网络正常                    ▼       │
│     延迟 45ms                          │
│                                         │
└─────────────────────────────────────────┘
```

- **位置**: 右上角固定

- **图标**: Wifi (满格)

## 动画效果详解

### 雷达扫描 (radar-sweep)

```css
@keyframes radar-sweep {
  0%   { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

/* 视觉效果 */
   ╱
  ╱ ← 扫描线从中心向外旋转
 ╱
```

### 脉冲呼吸 (status-pulse)

```css
@keyframes status-pulse {
  0%, 100% {
    opacity: 1;
    transform: scale(1);
  }
  50% {
    opacity: 0.8;
    transform: scale(0.95);
  }
}
```

### 离线闪烁 (offline-blink)

```css
@keyframes offline-blink {
  0%, 100% { opacity: 1; }
  50%      { opacity: 0.7; }
}
```

## 响应式设计

### 桌面 (> 768px)
- 宽度: 200px
- 圆角: 0.5rem (8px)
- 字体: 14px 正文 / 12px 辅助
- 图标: 16x16px

- 宽度: auto (自适应)
- 圆角: 0.5rem (8px)
- 字体: 12px 正文 / 10px 辅助

### 悬停效果
```

```

```
正常: 白色/50
```

## 可访问性

### 键盘导航
- `Tab`: 聚焦状态指示器
- `Enter/Space`: 展开/收起详情
- `Esc`: 关闭详情面板

### 屏幕阅读器
- 使用 `aria-label` 描述状态
- `role="status"` 标识状态区域
- `aria-live="polite"` 状态变化通知

### 对比度
所有文本与背景对比度 ≥ 4.5:1 (WCAG AA 标准)

## 设计决策

### 为什么选择雷达美学？

1. **语义契合**: 网络规划工具 = 信号监测 = 雷达
2. **专业感**: 航图/雷达 = 工业级应用
3. **动态感**: 扫描动画传达"正在监测"
4. **识别度**: 独特的视觉风格，易于记忆

### 为什么非阻塞式？

1. **不干扰工作**: 用户可以继续操作地图
2. **可关闭**: 用户可以主动忽略
3. **可重试**: 提供明确的操作选项

### 为什么使用深色主题？

1. **项目一致**: 整体应用使用深色主题
2. **降低眩光**: 减少长时间使用的视觉疲劳
3. **提升对比**: 状态指示器更加醒目
