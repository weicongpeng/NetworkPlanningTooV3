# License Generator 使用说明

## 功能介绍
`license_generator.py` 是一个交互式的许可证管理工具，用于生成和检查 Network Planning Tool 的许可证文件。

## 使用方法
运行程序后，会出现交互式菜单：

```
==================================================
Network Planning Tool - License Manager and Generator
==================================================
请选择操作:
1. 生成新许可证
2. 检查现有许可证
3. 生成并检查许可证
4. 退出
==================================================
```

根据菜单提示输入编号选择相应功能：

### 1. 生成新许可证
- 程序会提示输入许可证有效天数（默认30天）
- 生成加密的 license.dat 文件
- 显示许可证详细信息

### 2. 检查现有许可证
- 检查当前目录下的 license.dat 文件
- 验证许可证是否有效且未过期
- 显示验证结果

### 3. 生成并检查许可证
- 先生成新的许可证文件
- 然后立即验证生成的许可证
- 确保许可证可以正常使用

### 4. 退出
- 退出程序

## 运行方式
1. 通过命令行运行：
   ```
   python license_generator.py
   ```

2. 通过批处理文件运行：
   ```
   run_license.bat
   ```

## 注意事项
- 生成的许可证文件为 license.dat
- 请将 license.dat 文件放在 Network Planning Tool 可执行文件的同一目录下
- 许可证过期后需要重新生成
