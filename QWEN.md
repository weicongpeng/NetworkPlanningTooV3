# NetworkPlanningTool 项目说明

## 项目概述

NetworkPlanningTool 是一个综合性的无线网络规划工具，支持 LTE 和 NR (5G) 网络的 PCI (Physical Cell Identity) 规划、网络参数更新和邻区规划。该项目主要使用 Python 3 开发，结合了 GUI 界面和命令行功能，采用先进的覆盖圆算法进行邻区规划。

### 核心功能

1. **PCI 规划**：支持 LTE 和 NR 网络的 PCI 规划，包括基于距离的规划和双模约束
2. **邻区规划**：使用覆盖圆算法进行邻区规划，考虑了天线方向性、方位角等因素
3. **网络参数更新**：支持从现网工参压缩包中提取信息并更新全量工参文件
4. **GUI 界面**：提供直观的图形用户界面，便于操作和查看结果
5. **许可证管理**：包含许可证验证机制

### 技术栈

- **Python 3**: 核心编程语言
- **pandas**: 数据处理
- **numpy**: 数值计算
- **openpyxl**: Excel 文件处理
- **tkinter**: GUI 框架
- **cryptography**: 许可证加密

## 核心组件说明

### NetworkPlanningTool_V1.py

主程序文件，包含：

1. **NeighborPlanningTool 类**：邻区规划核心工具
   - 使用覆盖圆算法进行邻区规划
   - 支持覆盖圆距离系数(k)和覆盖圆半径系数(m)
   - 实现同站小区优先级最高的机制
   - 提供智能邻区选择算法和扇区分布优化

2. **PCIGUIApp 类**：GUI应用程序主类
   - 提供直观的图形用户界面
   - 包含PCI规划、参数更新、邻区规划等功能标签页
   - 支持鼠标悬停提示功能

3. **NetworkParameterUpdater 类**：网络参数更新工具
   - 从现网工参压缩包中提取信息
   - 更新全量工参文件
   - 支持模糊列名匹配

### test.py

基于覆盖圆算法的邻区规划测试脚本，包含原始的覆盖圆算法实现，用于算法验证。

### planning_lte_nr_enhanced.py

LTE/NR分离式PCI规划工具，支持：

1. 从待规划小区文件的LTE和NR工作表分别读取数据
2. NR小区使用mod30逻辑替代mod3逻辑
3. LTE和NR规划结果分别保存到不同文件
4. 优先确保最小PCI复用距离

## 算法特性

### 覆盖圆算法

NetworkPlanningTool 使用先进的覆盖圆算法进行邻区规划：

1. **k系数（站点到覆盖圆心距离系数）**：
   - 默认值为5/9(≈0.556)
   - 表示覆盖圆心位于主瓣方向距离站点(5/9)*Co处
   - k值越大，覆盖圆心离基站越远，方向性越明显
   - k值越小，覆盖圆心越接近基站，覆盖越接近全向

2. **m系数（覆盖半径系数）**：
   - 默认值为5/9(≈0.556)
   - 表示覆盖半径为(5/9)*Co
   - m值越大，覆盖圆半径越大，可获得邻区数量增多
   - m值越小，覆盖圆半径越小，邻区数量减少

3. **同站小区优先级**：同一站点的小区始终获得最高优先级

## 开发规范

### 代码结构
- 使用面向对象编程方法
- 每个功能模块封装在独立的类中
- 代码遵循Python PEP 8规范

### 许可证管理
- 项目包含许可证验证机制
- 通过license.dat文件验证授权状态
- 未授权用户无法使用完整功能

### 数据处理
- 支持Excel格式的工参文件
- 使用pandas进行数据处理
- 支持模糊列名匹配以适应不同格式的工参文件

4. **参数影响说明**：详细说明k、m系数对邻区规划的影响
5. **同站小区优先级**：确保同站小区获得最高优先级
6. **界面优化**：改进用户界面布局和交互体验
