# 数据管理页面视觉提示效果更新

## 更新内容

### 1. 导入数据过程视觉提示 ✅

为每个上传项添加了状态管理和视觉反馈：

**状态类型：**
- `idle` - 默认状态（灰色边框）
- `success` - 上传成功（绿色边框 + 绿色背景 + ✓ 图标）
- `error` - 上传失败（红色边框 + 红色背景 + ✗ 图标）
- `loading` - 上传中（灰色边框 + 旋转动画）

**视觉提示效果：**
- 上传成功时：边框变绿色，背景变浅绿色，显示"上传成功"文字和 ✓ 图标
- 上传失败时：边框变红色，背景变浅红色，显示"上传失败"文字和 ✗ 图标
- 状态持续 3 秒后自动恢复为默认状态

**实现方式：**
```typescript
// UploadArea 组件新增 status 属性
status?: 'idle' | 'success' | 'error'

// 根据状态动态设置样式
const getBorderClass = () => {
  if (loading) return 'border-muted bg-muted/50'
  if (status === 'success') return 'border-green-500 bg-green-50'
  if (status === 'error') return 'border-red-500 bg-red-50'
  return 'border-border hover:border-primary'
}
```

### 2. 已导入数据视觉提示 ✅

为已导入数据列表添加了更丰富的视觉提示：

**文件类型图标（带背景色）：**
- 地理化数据（扇区/点）：紫色背景 + 对应图标
- 全量工参：蓝色背景 + Excel 图标
- 待规划小区：橙色背景 + Excel 图标
- 现网工参：靛蓝色背景 + Excel 图标
- MapInfo 图层：青色背景 + 图层图标
- 其他 Excel：绿色背景 + Excel 图标
- ZIP 文件：灰色背景 + 上传图标

**文件类型标签（彩色圆角标签）：**
- 扇区数据 / 地理化（紫色）
- 全量工参（蓝色）
- 待规划（橙色）
- 现网工参（靛蓝色）
- MapInfo（青色）
- Excel（灰色）
- ZIP（灰色）

**路径记录状态：**
- 已记录路径的文件显示绿色"已记录"标签，带 ✓ 图标

**数据信息展示：**
- 显示数据内容摘要（如：LTE 10站/30小区 · NR 5站/15小区）
- 显示上传日期
