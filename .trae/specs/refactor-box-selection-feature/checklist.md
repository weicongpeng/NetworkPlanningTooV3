# Checklist

## 框选工具菜单
- [x] 框选按钮显示下拉菜单，包含"圆形"和"多边形"选项
- [x] 选择框选模式后菜单自动收起
- [x] 无可见图层时显示提示，不进入框选模式
- [x] 多图层可见时显示提示，不进入框选模式

## 圆形框选功能
- [x] 按下鼠标左键并拖拽可绘制圆形区域
- [x] 实时显示圆形边框
- [x] 释放鼠标后识别圆形区域内的扇区
- [x] 选中扇区边缘渲染为红色高亮

## 多边形框选功能
- [x] 点击地图添加多边形顶点
- [x] 实时显示多边形边框
- [x] 双击或右键完成多边形框选
- [x] 选中扇区边缘渲染为红色高亮

## 选中扇区红色高亮渲染
- [x] 选中扇区边缘渲染为红色（#ff0000），边框宽度为3
- [x] 高亮效果在地图缩放/移动后保持
- [x] 选中扇区自动置于图层顶层
- [x] 右键点击或点击清除按钮可清除高亮

## Ctrl+C 复制功能
- [x] Ctrl+C 可复制选中扇区的属性数据
- [x] 数据格式化为 TSV 格式，兼容 Excel
- [x] 复制成功显示提示信息
- [x] 输入框焦点时不干扰默认复制行为
- [x] 无选中数据时不执行复制

## 框选模式退出
- [x] Esc 键可退出框选模式
- [x] 退出后保留选中状态和高亮
- [x] 右键点击可清除选中状态

## 地图拖拽工具
- [x] 框选模式下显示"拖拽"按钮
- [x] 启用拖拽工具后可拖拽地图
- [x] 启用拖拽工具后点击不绘制框选区域
- [x] 禁用拖拽工具后恢复框选绘制功能

## 图层可见性控制
- [x] 框选模式下只能显示一个图层
- [x] 尝试开启多个图层时显示提示并阻止

## 代码质量
- [x] 框选逻辑已抽离到独立的 SelectionManager 类
- [x] 状态管理使用单一数据源
- [x] 代码注释清晰，逻辑易懂
- [x] 无 TypeScript 类型错误
